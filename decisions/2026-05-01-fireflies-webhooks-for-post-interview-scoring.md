# Use Fireflies webhooks to feed post-interview scoring

## Decision

Post-interview assessments will be triggered by Fireflies meeting/transcript webhooks rather than manual transcript copy-paste into Claude.

## Why

Theresa, Mariam, Michael and others all interview, so a per-interviewer manual flow does not scale; Fireflies supports the needed webhooks and already attaches candidate name to meetings, enabling automated post-interview report generation.

## Evidence

> Jehad Altoutou: It makes sense, honestly. And Fireflies does have webhooks. We can use these ones.

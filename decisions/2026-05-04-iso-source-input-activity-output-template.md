# Adopt Source/Input/Activity/Output as the standard ISO-style template for every pipeline stage

## Decision

Every stage of the AI tool pipeline (intake, enrichment, evaluation, sandbox, IT handover) must be documented using the same Source / Input / Activity / Output structure, with the contents of Activity free to vary per stage.

## Why

Bonaventure framed this as the non-negotiable ISO process shape; Michael accepted it as the wrapper. It standardises documentation across departments and lets the same shape carry forward as the company scales, and lets Bonaventure / Simon eventually align it with ISO 9001.

# Pilot the AIO Fireflies-to-Monday-Notion-Linear standup pipeline on IT/Ops first

## Decision

Before rolling the AIO standup-automation skill suite to other departments, the team will pilot it with IT/Ops, using Jehad's skills to update Monday, Notion and Linear from Fireflies transcripts.

## Why

IT/Ops is the closest collaborator and already juggling many balls; testing the pipeline with them first surfaces tailoring needs (the skills are not out-of-the-box portable) before promising it to every department. Once it works for IT/Ops, the same pattern scales to a company-wide knowledge base and agentic layer.

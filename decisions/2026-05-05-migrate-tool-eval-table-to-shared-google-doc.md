# Migrate the tool-evaluation ISO mapping table from Word to a shared Google Doc

## Decision

Simon will move the tool-evaluation table out of Word and into a shared Google Doc so everyone can edit it collaboratively and AI can fill it in.

## Why

Emailing Word documents back and forth lost version control; a shared Google Doc gives a single live artefact that humans and AI can both edit and that the team can iterate on.

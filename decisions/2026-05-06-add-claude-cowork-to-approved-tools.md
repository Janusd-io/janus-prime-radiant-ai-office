# Add Claude Cowork to the v3 approved-tools list

## Decision

Claude Cowork will be added to the v3 approved AI tools list, listed under Claude as 'Claude (including Cowork)' to distinguish it from Codex-style standalone coding apps.

## Why

Cowork ships inside the existing Claude plan and consumes no more tokens than the chat surface, so blocking it adds no cost control while preventing legitimate use. Bundling it under Claude (rather than as a separate entry) mirrors how Anthropic ships the product; Codex-equivalent split-app cases are treated separately because they are downloaded as distinct desktop apps.

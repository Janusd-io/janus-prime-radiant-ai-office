# GitHub is the skill library, treated as source code

## Decision

Skills will be stored, versioned and revision-controlled in GitHub, treated like source code with no ad-hoc edits.

## Why

Skills are software artefacts — GitHub already gives the company version control, audit trail, immutability of past revisions, and a controlled change pathway. Storing them anywhere editable would break ISO traceability. This also paves the way to publish vetted skills as company-wide Claude plugins later.

## Evidence

> Speaker 1: In this sense. We use GitHub for now because then GitHub's got a good version control. You can check it in.

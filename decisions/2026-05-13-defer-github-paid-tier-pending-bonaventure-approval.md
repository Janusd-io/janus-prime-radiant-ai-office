# Stay on GitHub free tier until Bonaventure approves the paid upgrade

## Decision

The Prime Radiant runs on GitHub's free tier for now; the ~$4/user/month paid upgrade (needed for branch protection and other controls) waits on Bonaventure's budget approval.

## Why

Branch protection and the other paid-tier controls are wanted but not blocking. Spending requires Bonaventure's sign-off, which has not been given, so the team is parking the upgrade rather than holding up the rollout.

## Evidence

> Speaker 3: We need to upgrade to a paid version. It's $4 per user per month... And Bonaventure's got them. You can't approve it, so we'll have to go in the free version for now.

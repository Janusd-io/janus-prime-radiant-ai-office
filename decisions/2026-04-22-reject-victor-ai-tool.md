# Reject Victor as an AI tool

## Decision

Victor is rejected because it lacks per-user access control inside shared Slack channels.

## Why

If a finance user is in a shared channel running Victor, every other channel member effectively inherits their financial-system access. The tool answers any participant's question with the running user's permissions, which is unsafe for cross-department rollout.

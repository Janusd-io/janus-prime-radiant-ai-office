# Markdown is the canonical interchange format for AI-marketing source material

## Decision

Project source files, downloads, and exchanged documents will default to Markdown so the LLMs do not waste tokens converting from PDF or DOCX.

## Why

Michael explained that LLMs were trained on Markdown and parse it natively, while PDFs and Word documents require pre-processing that consumes tokens and can lose structure (titles, headings). He enabled Markdown rendering in Andrew's Google account live during the meeting.

## Evidence

> Speaker 2: So make Markdown your lowest common denominator.

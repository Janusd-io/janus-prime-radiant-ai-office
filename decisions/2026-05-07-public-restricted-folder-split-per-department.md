# Each department Shared Drive splits into a public and a restricted folder

## Decision

Every department's Shared Drive will contain two top-level folders: a company-wide-readable one (named e.g. 'HR') and a department-restricted one (named e.g. 'HR private' / 'HR restricted').

## Why

Some department artefacts (brand guidelines, employee handbook, approved-tools registry) need to be readable across the whole company; others (insurance policy, financial statements, salary data) must stay inside the department. A two-folder split with group-based permissions implements this without complex per-file ACLs.

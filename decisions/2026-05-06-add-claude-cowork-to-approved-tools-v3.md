# Add Claude (including Cowork) to v3 approved tools list

## Decision

Cloud / Claude Cowork will be added to version three of the approved tools list, written as 'Claude' with 'including Cowork' in parentheses, kept distinct from Claude Code.

## Why

Cowork ships inside the standard Claude plan, blocking it produces no token savings, and there is no operational reason to exclude it once Claude itself is approved. ChatGPT splits Codex into its own app, but Claude bundles Cowork with the main product, so the registry should reflect that bundling. Final scope still gated by department-by-department need (marketing, finance, IT, engineering — not general purpose).

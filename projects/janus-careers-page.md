## Active Job Descriptions (as of 2026-06-10)

### AI Software Engineer — AI Office

Filed 2026-06-10. Full JD source: [[janus-ai-software-engineer-jd-2026]].

Key points:
- **Reporting line:** AI Office, reporting to Head of AI Office
- **Location:** Dubai, UAE — global operations base; travel to Singapore, Turkey, Spain, UK (15–25%)
- **Level:** mid-to-senior full-stack, 5+ years; banking/fintech/regulated-systems background prioritised
- **Core stack:** Python, TypeScript, React; AWS (primary); Docker/Kubernetes; MCP + multi-model orchestration (Claude + Gemini); Claude Code daily
- **Key responsibility:** Memory & Knowledge Systems — described as *"a structured, multi-graph, file-based system underpinning institutional intelligence across the company — among the hardest and most consequential engineering problems on the platform"* (i.e., the Prime Radiant / [[janus-prime-radiant-build]] architecture)
- **Mandatory application requirement:** Active GitHub profile with demonstrable commit history
- **Companion JD:** See also `Janus_Digital_AI Platform Engineer.docx` (in repo; not yet filed as source)

## Notes

The AI Software Engineer JD is the first role explicitly described as an AIO hire in this wiki. The Memory & Knowledge Systems section of the JD is effectively the public-facing description of the [[janus-prime-radiant-build|Prime Radiant]] engineering challenge — a notable signal that the institutional memory architecture is now part of Janus's external engineering identity.

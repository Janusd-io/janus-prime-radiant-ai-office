**PROCESS DISCOVERY TEMPLATE**

*Working Document — for process discovery sessions*

**What is a process?**

A process is a set of connected activities that turns inputs into
outputs. Every process has a clear starting point (what triggers it), a
clear ending point (what it produces), and defined steps in between.
Processes are performed by people, systems, or a combination of both.

**What is the purpose of a process?**

The purpose explains why this process exists — what problem it solves or
what value it delivers. A good purpose statement answers: "What would go
wrong if this process didn't exist?"

**What is a process owner?**

The process owner is the person responsible for making sure the process
works correctly and is kept up to date. This is not necessarily the
person who performs the process day to day.

**How to use this document**

Answer each question in your own words. There are no wrong answers — the
goal is to capture how the process actually works. Add comments wherever
something needs clarification.

| **BLOCK 5 — PERFORMANCE INDICATORS** |

A performance indicator is a simple measure that tells you whether this
process is working well. Think of it as an answer to the question: "How
do we know this process is achieving its goal?" A good indicator has a
clear target (e.g. within 2 days, no more than 10%) and can be measured
regularly. If you are unsure how to calculate an indicator, describe in
plain words what "good performance" looks like — we will work out the
formula together.

|  |  |  |  |  |
|----|----|----|----|----|
| **\#** | **What do we measure?** | **What is the target?** | **How is it calculated?** | **Source Document** |
| *1* | *How quickly the final report is delivered to the client after the project is completed* | *Within the agreed deadline* | *Number of reports delivered on time ÷ total reports delivered × 100%* |  |
| *2* | *How often the client requests revisions after receiving the report* | *No more than 10% of projects* | *Number of revision requests ÷ total reports delivered × 100%* |  |

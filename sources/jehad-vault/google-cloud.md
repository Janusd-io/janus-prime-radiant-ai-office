# Google Cloud

Google's enterprise cloud arm. Distinct from "Google" generally — this entity covers the GCP product surface specifically: BigQuery, Vertex AI, Gemini API, GKE, Cloud Run, the data catalog, and the broader agent-ecosystem moves Google has been making at speed in 2026. Existing vault page `gcp.md` is the same vendor under a less-descriptive slug; flagged for dedup.

## Notable 2026 launches relevant to this wiki

- **Agentic Data Cloud** (2026-04-22) — Cloud Next 2026. Knowledge Catalog + cross-cloud lakehouse + Data Agent Kit. Reframes BigQuery for agent-scale workloads.
- **Official Agent Skills Repository** (2026-04-22) — same Cloud Next event. Condensed expertise for GCP products as agent skills.
- **A2A protocol push** — Google is the most visible advocate for cross-vendor agent-to-agent protocol standards.

## Posture

Sharper agent-platform play than expected from Google a year ago. The "human scale → agent scale" framing (Andi Gutmans, VP/GM Data Cloud) gives an organising story for the data + skills + protocol announcements landing together.

## Janus relevance

Currently low active dependency at Janus (no production GCP workloads as of 2026-05-06). Worth tracking primarily as a reference point against which Anthropic / OpenAI / AWS positioning gets evaluated, and as an MCP / A2A standards bellwether.

## Watch for

- Whether AWS / Azure ship analogous "agentic data cloud" stories.
- Skill format compatibility: Google skills vs. Anthropic skills vs. ecosystem fragmentation.
- Pricing for the Data Agent Kit when it leaves preview.

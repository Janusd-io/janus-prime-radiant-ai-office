## Attio product overview (from Moses Francis)

- **Core proposition:** Not just a CRM — a flexible relational data model. Track any dataset, gain visibility, automate manual work.
- **Custom objects:** Map any data with integrations or API connections. Examples: Xero read/write API, Linear integration.
- **Workflows:** Automate anything previously manual (e.g., record from website → create person record → tie to Xero contract).
- **Call transcription built in:** Records calls, parses to the relevant record, allows interaction (send emails with full context). Could replace Fireflies for meeting transcription.
- **AI data analyst:** SQL write tools, enable custom tools, AI chat over records.
- **Lead-to-closed-won pipeline:** Forms, automations, full revenue engine claim.

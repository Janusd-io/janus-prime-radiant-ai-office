# Move the evaluation framework table from Word to a shared Google Doc

## Decision

Simon will lift the evaluation framework table out of the Word document and into a shared Google Doc so all three (Simon, Speaker 1, Speaker 2) can edit it collaboratively in real time.

## Why

Emailing Word documents around loses version control and forces sequential edits. A shared Google Doc gives a single living artefact that AI tools can also read and help fill in, which matches the broader 'one source of truth' principle being applied to skills.

## Evidence

> Speaker 2: Take that table out of Word. Put it in the shared Google Doc. And we can all access it together.

# Garry Tan

**Role:** President of Y Combinator
**Organisation:** Y Combinator (YC)

Garry Tan is the President of Y Combinator, previously a partner there and co-founder of Posterous (acquired by Twitter). He is in this wiki primarily because of his open-source markdown-first agent memory layer ([[gbrain]]), which demonstrates the same architectural discipline as the Janus Prime Radiant pattern applied to a personal AI chief-of-staff use case.

## Why he's in this wiki

Garry Tan published **gBrain** — an open-source markdown-first memory harness for AI agents. The design: structured Markdown files as the persistent memory substrate, with an LLM as the memory supervisor. This is the same architectural axis as the Janus Prime Radiant system (file-based, structured, LLM-supervised knowledge). The [[gbrain]] vendor page documents the pattern.

The "YC / Tan memory layer" framing appears in [[2026-05-22-gbrain-yc-tan-memory-layer]] as one of the external-validation signals for the Prime Radiant methodology. Garry Tan publishing a markdown-first personal knowledge system as YC President is significant: it signals that the pattern Janus is using as institutional infrastructure is the same pattern the most prominent tech accelerator's leadership uses for personal infrastructure. Validates the methodology without being a direct competitor.
